# Blood Donation Management System

A comprehensive blood donation and management platform with role-based dashboards.

## Features

- **Admin Dashboard**: Complete control center with donor management, inventory overview, and system controls
- **Hospital/Blood Bank Dashboard**: Inventory management, emergency requests, and donor matching
- **Donor Dashboard**: Profile management, donation history, and emergency alerts
- **Requester Dashboard**: Create requests, track status, and contact information

## Roles

1. **Admin** - System administrator with full control
2. **Hospital/Blood Bank** - Medical facilities managing inventory
3. **Donor** - Blood donors managing their profile and donations
4. **Requester** - Patients/attenders requesting blood

## Quick Start

```bash
npm install
npm run dev
```
